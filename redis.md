# Redis #
## Redis 概述 ##

   Redis是基于ANSI C语言编写，接近汇编语言的机器语言，运行十分快速，其次它是基于内存的读/写，速度自然比数据库的磁盘读/写要快的多，而且相比一般的NOSQL支持的数据类型又比较多，数据结构较为简单，规则较少，因此很受互联网的欢迎
   
   Redis在JavaWeb中一般两个使用场景，一个是缓存，另一个是需要高速读/写的场合
   
## Redis 的6种数据结构##

- STRING(字符串)
   
- LIST(列表)
   
- SET(集合)
   
- HASH(哈希散列表)
   
- ZSET(有序集合)
   
- HyperLogLog(基数)
